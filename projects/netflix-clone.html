Au cœur du projet

Analyse approfondie de l'architecture et des solutions implémentées.

Vue d'ensemble

Ce projet est une application web Single Page Application (SPA) réalisée avec React et TypeScript. L'objectif était de construire une architecture saine capable de gérer l'authentification utilisateur, le paiement d'abonnements récurrents et la diffusion de contenu multimédia.

Challenges & Solutions

Authentification & Sécurité

Mise en place d'un système robuste avec JWT (JSON Web Tokens) et cookies HTTP-only pour prévenir les failles XSS. Hashage des mots de passe avec Bcrypt.

Paiements (Stripe)

Intégration de l'API Stripe pour gérer les souscriptions mensuelles. Utilisation des Webhooks pour synchroniser le statut en temps réel.

Performance

Optimisation du chargement des images (Lazy Loading) et mise en cache des requêtes API pour assurer une fluidité maximale.

Stack Technique

React TypeScript Node.js Express MongoDB Tailwind

Fonctionnalités Clés

  • Authentification complète (Inscription, Connexion)
  • Plans d'abonnement (Basic, Standard, Premium)
  • Catalogue de films par genres
  • Recherche instantanée
  • Liste de favoris ("Ma liste")
  • Lecture vidéo fluide
